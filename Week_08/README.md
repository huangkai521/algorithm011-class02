## 位运算
#### 1.异或^
+ 规则：相同为0，不同为1。也可用“不进位加法”来理解。
+ 异或特点：
	+ x^0 = x
	+ x^x = 0
	+ x^1s = ~x // 1s = ~0
	+ x^~x = 1s
	+ c = a^b => a^c = b, b^c = a // 交换两个数
	+ a^b^c = (a^b)^c  = a^(b^c)
+ 指定位置的位运算：
	+ 将x最右边的n位清零：x&(~0<<n)
	+ 获取x的第n位值(0或者1)：(x>>n)&1
	+ 获取x的第n位的幂值：x&(1<<n)
	+ 仅将第n位置为1：x|(1<<n)
	+ 仅将第n位置为0：x&(~(1<<n))
